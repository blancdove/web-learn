# DOM节点树



### 节点树

+ parentNode

	父节点（最顶端的parentNode为 #document）

+ childNodes 

	子节点们

+ firstChlid

	第一个子节点

+ lastChild

	最后一个子节点

+ nextSibling

	后一个兄弟节点

+ previousSibling

	前一个兄弟节点



### 元素节点树

+ parentElement

+ children

	子元素节点们

+ fristElementChild

+ lastElementChild

+ nextElementSibling

+ previousElementSibling



### 节点的四个属性

1. nodeName

	节点名

2. nodeValue

	text 节点或 comment 节点的文本内容，可读写

3. nodeType

	节点类型

	+ 元素节点      1
	+ 文本节点      3
	+ 注释节点      8
	+ document       9
	+ DocumentFragment       11

4. attribute

	Element节点的属性实时集合
